ğŸ” Template Syntax Explained

Notice the special syntax in this HTML file:

  • {{ "{%" }} for item in list %} - Loop through data (like Python's for loop)
  • {{ "{{" }} variable }} - Display a variable's value
  • {{ "{%" }} if condition %} - Conditional logic (like Python's if statement)
  • {{ "{%" }} extends "base.html" %} - Inherit from another template

This is Jinja2 template syntax. It lets you write Python-like code in your HTML to make it dynamic!
